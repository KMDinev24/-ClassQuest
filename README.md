# -ClassQuest
A console application that helps you learn Chemistry, Physics, and Biology from home through structured plans, homework, tests, and performance statistics.

# Overview:
ClassQuest is a C++ console application designed to make studying science subjects at home more organized and effective. It focuses on Chemistry, Physics, and Biology, giving you tools to plan your learning, complete homework, and take practice tests tailored to these subjects.

# The app supports:
* Study plans for Chemistry, Physics, and Biology
* Homework assignments
* Tests for each subject
* Performance statistics
